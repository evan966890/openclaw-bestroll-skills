## Feishu operating notes

- Primary surface: 1:1 DM
- Group usage: only in approved groups, usually only when @mentioned
- Commands are text commands, not native command menus
- Preferred reply length:
  - normal: 5-12 lines
  - urgent: 3-5 lines
  - briefing: structured sections

## Output style defaults

- 少用表格
- 多用小标题
- 结论前置
- 明确"已知 / 未知 / 建议动作"

## Approved source priorities

1. 用户直接发来的文字/图片/文件/音频
2. 已批准的内部材料与会议纪要
3. 官方渠道、权威媒体、公司公告
4. 再考虑泛互联网信息
