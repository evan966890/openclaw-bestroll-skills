# 技能：兴趣画像建立与维护

## 触发条件

首次使用知识管家时自动触发；或用户说"更新我的关注方向"。

## 执行步骤（交互式对话）

1. 问用户："你现在最想持续了解哪些方向？"
2. 根据回答，追问具体维度：
   - 关注深度：泛泛了解 / 深度跟踪 / 研究级
   - 关注维度：技术突破 / 商业应用 / 政策变化 / 行业格局 / 投资机会
   - 信息源偏好：中文为主 / 英文为主 / 都要
3. 问推送偏好：
   - 频率：每天 / 每两天 / 每周
   - 时间：早上 / 中午 / 晚上
   - 条数：精简（3条）/ 适中（5条）/ 丰富（7条）
4. 生成 `radar/INTERESTS.md`

## 输出格式（radar/INTERESTS.md）

```markdown
# 知识雷达 · 兴趣画像

## 活跃主题

### [主题名称]

- 关键词：[搜索用关键词列表]
- 深度：泛泛了解 / 深度跟踪 / 研究级
- 维度：[技术/商业/政策/行业/投资]
- 语言：[中/英/双语]
- 上次互动：[日期]
- 状态：活跃 / 降温 / 暂停

## 推送偏好

- 频率：每天
- 时间：早上8:00
- 条数上限：5
- 信息源偏好：优先权威来源

## 排除项

- [用户明确说不感兴趣的方向]
```

## 维护规则

- 用户连续3次对某主题选择"阅后即焚" → 建议降温
- 用户新增兴趣点 → 即时更新
- 每月主动问一次："你的关注方向有变化吗？"
