# Extraction Rules

## 先识别什么算行动项

必须同时满足至少 2 项：

- 有明确动作
- 有明确对象
- 有明确结果
- 有时间要求

## 常见伪行动项

以下默认不算：

- “关注一下”
- “看一眼”
- “后面再说”
- “尽快推进”
- “大家同步一下”

这些只能作为背景，除非上下文补足 owner / deadline / 完成标准。

## 提取顺序

1. 动词：做什么
2. owner：谁负责
3. deadline：何时完成
4. 完成标准：做到什么算完成
5. 阻塞：卡在哪里
6. 下一步：下一次推进动作

## 多人聊天处理

- 只因为某人说话，不代表他是 owner
- 只有明确承接动作的人才记为 owner
- 如果只有“你来跟一下”这类话，owner 默认待补
