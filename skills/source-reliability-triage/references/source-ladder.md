# Source Ladder

## 可信度分层

### 高

- 公司公告
- 官方新闻稿
- 监管披露
- 原始文件 / 原始数据
- 一手当事人公开表述

### 中

- 权威媒体报道
- 行业研究机构
- 记者明确署名并给出信源
- 交叉验证后的二手整理

### 低

- 未署名搬运
- 社交平台截图转述
- “据说”“听说”“群里看到”
- 明显标题党、旧闻翻炒、断章引用

## 动作矩阵

- 高可信 + 高相关：推送或转分析
- 高可信 + 低相关：可归档，不主动推
- 中可信 + 高相关：标注不确定后可提示
- 低可信：默认不推，不归档

## 红旗信号

- 没有时间
- 没有原始链接
- 数字没有出处
- 引述对象模糊
- 内容情绪强但事实弱
