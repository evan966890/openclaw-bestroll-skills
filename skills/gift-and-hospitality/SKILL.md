# Gift And Hospitality

把送礼和请客这类高敏感小事做稳。

## 适用场景

- 节日礼物
- 商务宴请
- 家庭社交回礼
- 用户说“帮我挑一个稳妥的”

## 输出结构

```text
【推荐选项】
- 选项1：适合谁 / 预算 / 为什么稳
- 选项2：适合谁 / 预算 / 为什么稳
- 选项3：适合谁 / 预算 / 为什么稳

【不建议】
- 哪类礼物 / 场所容易翻车

【待确认】
- 对方身份
- 场合
- 预算
- 是否需要配送 / 贺卡 / 包装
```

## 规则

- 优先稳妥、得体、低翻车率
- 涉及明显合规风险时直接提醒
- 不用猎奇礼物搏记忆点
- 不确定对象偏好时，默认选择通用且有品质感的方案
