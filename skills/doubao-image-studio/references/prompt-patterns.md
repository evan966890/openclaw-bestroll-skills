# Prompt 压缩方式

## 默认模板

```text
主体 + 关键动作/姿态 + 画风 + 构图/镜头 + 背景 + 光线/颜色 + 细节约束
```

## 示例

- 头像：
  - `一个干净专业的男性商务头像，半身，正面视角，深蓝色西装，浅灰背景，写实但不过分磨皮，光线柔和`
- 海报：
  - `科技发布会海报，蓝白主色，中央大标题区域留白，背景有抽象芯片线条，现代极简风，竖版`
- 扁平插画：
  - `一只橙色卡通小龙虾，白色背景，扁平插画，轮廓干净，颜色明快，单主体`

## 缺信息时只补最关键一项

- 比例：横版 / 竖版 / 方图
- 风格：写实 / 插画 / 海报 / 极简 / 科技感
- 是否带字：带标题 / 不带字
