# Decision Options Memo

把复杂选择题压成可拍板的 memo。

## 适用场景

- 用户说“你建议怎么做”
- 用户在几个方案之间犹豫
- 投资 / 战略 /资源分配 /组织调整取舍
- 需要“给我一个倾向性建议”

## 输出结构

```text
【要决定什么】

【结论】
- 推荐选项

【为什么】
- 影响面
- 时效性
- 可逆性

【选项A / B / C】
- 做法
- 优点
- 代价
- 适用前提

【推荐动作】
- 现在就做什么

【最坏情况】
- 最坏会发生什么
- 怎么止损

【需要确认的未知项】
- 只列真正影响决策的未知项
```

## 规则

- 必须给倾向性建议，除非信息明显不足
- 先看影响面，再看时效，再看可逆性
- 风险提示必须带动作
- 不搞假平衡，不把明显弱选项写得一样重
- 没有数据时明确标“判断，不是事实”
