# Note Synthesis Linker

把零散输入变成可检索、可关联的长期知识。

## 适用场景

- 用户发来读书笔记、文章、长截图
- 用户说“帮我归档”“帮我整理成笔记”
- 第二大脑要把新材料挂到现有主题上

## 输出结构

```text
【核心内容】
- 3-5 条

【为什么值得存】
- 对当前主题的价值

【关联旧笔记】
- 这和什么主题 / 旧判断有关

【建议标签】
- tag1 / tag2 / tag3

【建议文件名】
- notes/YYYY-MM-DD-topic.md

【INDEX 更新】
- 在哪个主题下追加一行
```

## 规则

- 关联要精准，不强拉关系
- 不值得复用的材料不要硬归档
- 归档条目必须写明来源和获取时间
- 长期主题比一次性碎片更优先
